## 📋 About The Project

**Super Shanghai Dry Cleaners** is a comprehensive laundry management system designed to streamline operations for dry cleaning businesses. Built with Laravel and modern web technologies, it provides an intuitive interface for managing bookings, customer data, deliveries, payments, and generating detailed reports.

### 🎯 Key Objectives

- ✅ Streamline booking and order management
- ✅ Track customer information and history
- ✅ Manage deliveries (full and partial)
- ✅ Handle payments and generate financial reports
- ✅ Real-time shop invoice management
- ✅ Generate professional business reports

## ✨ Features

### 👥 Customer Management

| Feature | Description |
|---------|-------------|
| Customer Registration | Add new customers with name, mobile, and address |
| Customer Codes | Assign unique codes to customers |
| Search & Filter | Live search by name, mobile, or customer code |
| Customer History | View all bookings, payments, and delivery history |
| Edit/Delete | Update or remove customer information |

### 📦 Booking Management

| Feature | Description |
|---------|-------------|
| POS Booking | Create bookings with multiple items |
| Category Selection | Male, Female, Child, Household categories |
| Cloth Types | Predefined cloth types with auto price |
| Color Selection | Multiple color selection with quantity |
| Delivery Options | Normal, Same Day, Next Day, Urgent, 4/5 Days |
| Expected Date | Auto-calculated based on delivery type |
| Invoice Generation | Automatic unique invoice numbers |

### 🚚 Delivery Management

| Feature | Description |
|---------|-------------|
| Full Delivery | Complete order delivery in one go |
| Partial Delivery | Deliver items in multiple batches |
| Receiver Info | Optional receiver name and mobile |
| Delivery Notes | Add special instructions |
| Undeliver | Reverse delivery if needed |
| Delivery History | Track all deliveries per booking |

### 💰 Payment Management

| Feature | Description |
|---------|-------------|
| Partial Payment | Accept partial payments |
| Multiple Methods | Cash, Card, Bank Transfer, Online |
| Payment History | Track all payments per booking |
| Due Tracking | Automatic due amount calculation |
| Payment Status | Full Pay, Partial Pay, Full Due |

### 📊 Reports

| Report Type | Description |
|-------------|-------------|
| Booking Report | All bookings with filters (date, status) |
| Delivery Report | Full and partial delivery reports |
| Urgent Orders | Track urgent delivery orders |
| Non-Delivered | Items pending delivery |
| Short/Long Bookings | Based on item quantity (≤3 or ≥15 items) |
| At Shop Report | Shop invoices (excludes delivered items) |
| Revenue Analysis | Financial reports |
| Saved Reports | Auto-save all generated reports |

### 🏪 Shop Management

| Feature | Description |
|---------|-------------|
| Invoice Ranges | Generate invoice number ranges |
| Regular/Extra | Mark invoices as regular or extra |
| Missing Items | Flag missing invoices |
| Status Tracking | In Shop, Delivered, Pending, Partial |
| Search/Filter | Search by range, date, status, type |
